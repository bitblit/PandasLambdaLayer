
# Setting Up
1. **python3 -m venv ./venv**
2. Switch to your virtual env: **source venv/bin/activate**
3. **pip3 install -r requirements.txt**

# Quitting the current virtual enviroment
**deactivate**

# Deploying to AWS
4. **./deploy.sh**
